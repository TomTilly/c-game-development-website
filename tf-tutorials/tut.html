C Programming Language Basics

1. Variables

A variable is used to store information. Each variable...

  1. stores a particular kind of data (its type).
  2. has a name that you give it.

1.1 Variable Declaration

You must declare a variable before you use it. A variable's declaration tells the program what it's type and name are. Let's make a variable to store the player's score:

int score;

This variable is declared to be of type int (integer) and named score. Each statement in a program must be terminated with a semicolon.

1.2 Variable Assignment

To store a value in a variable use the assignment operator, =.

score = 0;

As a shorthand, you can put the declaration and assignment on a single line:

int score = 0;
