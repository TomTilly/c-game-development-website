C Programming Language Basics

3. Program Structure

Your program begins within the main function.

int main()
{
    printf("Hello, world!\n");
    return 0;
}

main is a special function, this is where every C program begins. When you run your program, the system calls main in order to execute it. main returns an int. Returning "0" tells the system the program terminated normally.

The above program prints the message "Hello, world!" to the screen. Create a text file named "hello.c", open it, and type in the above program.

To run the program, you need to compile it, that is, cover the code into instructions your computer can understand and execute. At the command line, type "gcc hello.c". If all goes well, there should now be a file called "a.out". Any errors are reported at the command line. Fix them and try again. 'a.out' is the exectable for your program. You can double click this program to run it.
