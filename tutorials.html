C Programming Language Basics

2. Functions

A function is a block of code that performs a task.

Here's the definition of a function called "ProcessGame":

ProcessGame()
{
    UpdatePlayer();
    MoveEnemies();
    CheckForCollisions();
}

The function's name is as the top, and is always followed by a pair of parentheses. The lines between each brace are the body of the function. These lines get executed whenever you use ProcessGame. To use MyFunction, you write its name followed by a pair of parantheses. This is called "calling the function":

ProcessGame();

At this point in the code, the contents of ProcessGame will be executed, that is, UpdatePlayer (another function) is called, then MoveEnemies, then CheckForCollisions.

2.1 Passing Data to a Function

Sometimes you will need to send a function some data. You put the value or values you want to pass into the function in the parentheses. E.g.:

IncreaseScoreBy(10);

The definition of IncreaseScoreBy might look like this: 

void IncreaseScoreBy(int amount)
{
    score += amount;
    UpdateScoreText();
    CheckIfWon();
}

Inside the parentheses you see what looks like a variable declaration. As part of the function definition, this tells the program that IncreaseScoreBy takes an integer value. This value is called a parameter. You can see that the value passed in above (10) is used in the function as the variable amount, and is added to the player's score. After this, two other functions are called that perform other actions.

2.2 A Function's Return Value

Just like a variable, a function can have a value. This is its return value The following is an example of a function that has a value of '5': 

int Five()
{
    return 5;
}

The int before the function name tells the program this function returns an integer. Note that this example is just for instructive purposes! You would never actually need to create or use a function like this, since you can just use the number '5', as is! So let's look at a real example!

// Returns the number of points received at end of level
int EndLevelPoints()
{
    int points;

    points = 0;
    points += numgems * 30;
    points += timeleft / 2;
    points -= enemieleft * 3;

    return points;
}

The '//' begins a comment. Everything on this line afterward is ignored by the program. This is used to add descriptions to your code. In this case, it's used to clarify what this function does.

At the beginning of the function, a variable points is declared, and then initialized (i.e., assigned a value for the first time) to 0. After this is added various amounts depending on what occurred during the level. Finally, the total points is returned. Let's look how this might be used.

int endpts;

endpoints = EndLevelPoints();
printf("You scored %i points this level!\n", endpoints);
score += endpoints;
printf("Your total score is: %i\n", score);
